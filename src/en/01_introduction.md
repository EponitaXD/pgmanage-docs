## Supported databases

- PostgreSQL
- SQLite
- MySQL
- MariaDB
- Oracle

## Supported Platforms

 - Linux
 - Windows
 - Mac OSX

**Note:** PgManage currently only supports x86_64 architecture. ARM64 support may be added in the future
